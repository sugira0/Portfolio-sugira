## üöÄ Deployment Options

### Option 1: Netlify (Recommended)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop your project folder
3. Your site will be live instantly!

### Option 2: GitHub Pages
1. Push code to GitHub repository
2. Go to Settings > Pages
3. Select source branch (main)
4. Your site will be available at `username.github.io/repository-name`

### Option 3: Vercel
1. Go to [vercel.com](https://vercel.com)
2. Import your GitHub repository
3. Deploy with one click

## üìù Customization

### Adding Your Images
Replace placeholder images in `assets/images/`:
- `sugira.jpg` - Your profile photo
- `projects/web1.png` - Legacy Heritage Tours screenshot
- `projects/boom.png` - Boom Travel screenshot
- `projects/softlabs.png` - SoftLabs screenshot

### Updating Contact Form
The contact form uses Formspree. Update the form action in `index.html`:
```html
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
```

- Update personal information in the About section
- Add your actual project links
- Update social media links with your profiles
